
## 工具集成类项目

记录不错的工具类及优秀的代码片段，提供研发过程是简化重复工作

### 目录结构描述
```
├── app                         // 应用服务
│   ├── generator               // 自动生成代码
│   ├── project                 // 应用服务代码
│   ├── Application             // 应用服务启动类
├── code                        // 业务代码设计
│   ├── design                  // 设计模式
│   ├── reusability             // 可复用代码
├── config                      // 配置
├── ratelimter                  // 限流
├── utils                       // 工具包
│   ├── generator               // mybaits-plus自动生成工具
│   ├── tools.*                 // 工具类
├── readme.md                   // 说明文档

```
